Packagebangfe.display.container
Classpublic class TransitionContainer
InheritanceTransitionContainer CoreDisplayObject flash.display.Sprite

This is a simple transition container. Add a display object via addChild and the child will be transitioned on. Old children will be transitioned off in an OUT then IN transition. For detailed transitions implement the ITransitionable interface. If an item implements ICoreObject, it's destroy method will be called instead of direct removal. Check the examples for usage
